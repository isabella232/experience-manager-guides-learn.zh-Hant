# 建立和預覽主題 {#id21BQE0RJ042}

AEM Guides隨附強大的功能，可從Assets UI建立DITA主題。 然後，您可以從Assets UI預覽主題。 「預覽」模式也提供許多強大的功能，可協助您執行關鍵作業，例如檢視中繼資料、在Web編輯器中編輯主題、開啟原始碼以及尋找網頁型版本差異。
