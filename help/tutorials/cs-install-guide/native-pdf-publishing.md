# 原生PDF發佈

使用AEM Guides，您可以使用原生PDF輸出產生功能來產生個別主題的PDF或對應檔案。 產生PDF輸出時，您可以自訂並使用PDF範本。 您可以建立並設計頁面配置，以根據您的需求在PDF輸出中顯示各種區段。

此外，AEM Guides也支援語言變數，協助您定義當地語系化字串。 它也可讓您為PDF輸出中存在的各種元素定義樣式。 如果您是進階使用者，也可以直接使用CSS來建立進階頁面配置。
